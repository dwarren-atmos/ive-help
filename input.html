This command reads and executes an IVE log file.

Widget:

  • From the IVE Main Window, activate the Input Menu by clicking on "Input" with the left mouse button. From the Input Menu, select "Input Commands" to bring up the Input Window. Now, simply specify the path and name of the logged IVE session you wish to reproduce.
Command:
  • INPUT=filename
Arguments:
  • filename: the path and name of the IVE log file.
Examples:
  • INPUT=~/ive/log/session1.log
Notes:
  • Procedures that you often repeat in IVE sessions can be logged using the LOGFILE command and subsequently reproduced each time with the INPUT command. This allows you to save time during routine applications.
