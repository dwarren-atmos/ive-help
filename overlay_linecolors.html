This command changes the order of colors for each successive color contour overlay.

Widget:

  • There is no widget interface for this command.
Command:
  • OVERLAY_LINECOLORS=color1, color2, color3, ..., colorN (where N is the number of colors in the color sequence you have chosen).
Arguments:
  • Possible colors are red, blue, green, violet, brown, black, white, and yellow. The order of these arguments determines the color of each successive color contour overlay.
Examples:
  • OVERLAY_LINECOLORS=red, brown, yellow, green, blue, violet, black
Notes:
  • To change the order of line patterns, see OVERLAY_PATTERNS.
  • The default color overlay sequence is as follows: red, green, violet, brown, black, blue.
