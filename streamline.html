This command specifies whether or not a vector plot will be displayed as streamlines or vectors.

Widget:

  • From the vector Properties Window, select either "Vectors" or "Streamlines" with the left mouse button.
Command:
  • STREAMLINE=[ON / OFF]
Arguments:
  • ON - all successive vector plots displayed as streamlines
  • OFF - all successive vector plots displayed as vectors
Examples:
  • STREAMLINE=ON
Notes:
  • Detailed instructions for creating vector plots in IVE are contained in Producing a Vector Plot.
