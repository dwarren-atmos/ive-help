With this command, contour plots can be produced which are overlayed on map projections.

Widget:

  • Select "Base Map" from the "Plot Background" menu. When the Map Optons widget appears, use the left mouse button to click on the Draw Map toggle button. The value of this button will toggle between YES (the plot will be overlayed on a map background) and NO (the plot will be displayed without a map background).
Command:
  • DRAWMAP = choice
Arguments :
  • choice: choice may be either ON or OFF. If choice = ON, all subsequent plots will be displayed on the map background specified by the command MAPROJ. If choice = OFF, all subsequent plots will be displayed without a map background.
Examples :
  • DRAWMAP = OFF
    Plots are displayed without a map background.
Notes :
  • if a map projection has not previously been selected, the plot will be drawn on a Cylindrical Equidistant map projection. this option remains in effect until changed
