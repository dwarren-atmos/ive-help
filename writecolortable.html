This command saves the current color table to your directory in *.ctb format.

Widget:

  • There is no widget for this command.
Command:
  • WRITECOLORTABLE=filename
Arguments:
  • filename: the full pathname of the file where the color table will be stored in *.ctb format.
Examples:
  • WRITECOLORTABLE=~/color/roygbv.ctb
Notes:
  • The default color table directory can be changed using the environment variable IVECTB.
