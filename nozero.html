This command can be used to turn off the zero contour line.

Widget:

  • Click your left mouse button on the Plot Zero Contour button in the "Properties" window. The value of this button will toggle between Yes and No.
Command:
  • NOZERO = [ON / OFF]
Arguments :
  • ON - the zero line is drawn on contour plots.
  • OFF - the zero line is not drawn on contour plots.
Examples :
  • NOZERO = OFF
Notes :
  • The default is on.
  • This option remains in effect until changed.
