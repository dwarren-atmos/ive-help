This command allows you to visually compare future plots with the current one by creating an independent graphics display window containing the current image.

Widget:

  • From the IVE Main Window, click on "Save Window" with the left mouse button.
Command:
  • SAVE_WINDOW
Examples:
  • SAVE_WINDOW
Notes:
  • The SAVE_WINDOW command is not to be confused with DUMP. SAVE_WINDOW copies your plot to a new window, but DUMP copies it to a graphics file.
