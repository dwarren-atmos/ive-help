This command switches between the normal plot setting and the movie setting.

Widget:

  • There is no widget interface for this command.
Command:
  • MOVIE_LABELS=[ON / OFF]
Arguments:
  • ON - all successive plots are created in the movie setting
  • OFF - all successive plots are created in the normal plot setting
Examples:
  • MOVIE_LABELS=ON
Notes:
  • The movie setting creates plots with black backgrounds and white axes. The tic labels and axis labels are enlarged and displayed in white as well. The movie setting is useful for filming ive plots and loops.
