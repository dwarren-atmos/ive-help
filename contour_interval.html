This command makes it possible to choose a plot contour interval. If a value > zero is chosen, that contour interval will be used. If a value <= 0 is chosen, the plotting routines will choose a contour interval.

Widget:

  • Click the left mouse button on the Interval button. Move the cursor to the type-in box and enter the desired contour interval, then press <RETURN>.
Command:
  • CONTOUR_INTERVAL = interval
Arguments :
  • interval: interval is the desired contour interval
Examples :
  • CONTOUR_INTERVAL = 0
    The contouring routines choose a contour interval.
