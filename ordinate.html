This command allows you to set the minimum and maximum values on the plot ordinate.

Widget:
  • In the Line Options widget, move to the Vertical Axis type-in boxes and type in your selection. Press <RETURN> when finished. Note that either the ordinate minimum, or maximum, or both may be set. If either of these values is set to default then IVE will choose an appropriate value.
Command:
  • ORDINATE = min,max
Arguments :
  • min: the minimum value along the plot ordinate.
  • max: the maximum value along the plot ordinate.
Examples :
  • ORDINATE = -5,5 the plot ordinate will run from values of -5 to 5. ORDINATE = ,2 the graphics package will choose an appropriate minimum value (provided the user has not previously set it), and the maximum value will be 2.
Notes :
  • This option remains in effect until changed.
  • Setting either the minimum or maximum to default means that the graphics package will choose an appropriate value.
  • See also VERTICAL SCALE.
