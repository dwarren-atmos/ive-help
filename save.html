This commands lets you keep a plot up for subsequent overlays.

Widget:

  • There is no widget for this. It is automatically done by the OVERLAY_PLOT command.
Command:
  • SAVE = [ON / OFF]]
Arguments :
  • ON - do not clear prior to the next plot
  • OFF - do clear prior to the next plot
Examples :
  • SAVE = OFF
Notes :
  • This command should not be entered by the user. IVE uses this command internally to handle Overlay Plots.
