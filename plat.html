This command specifies the projection latitude for your map projection.

Widget:

  • There is no widget interface for this command.
Command:
  • PLAT=latitude
Arguments:
  • latitude: the desired projection latitude between -90 and 90 degrees.
Examples:
  • PLAT=45
    Designates 45 degrees north as the projection latitude.
Notes:
  • To set the projection longitude, refer to PLON.
  • To set the map rotation, refer to ROTA.
  • For more information on map projections, refer to the manual for NCAR Graphics subroutine "Easy Map".
