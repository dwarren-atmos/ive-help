Using the FILLTYPE command, you can produce either contour line plots or color-filled plots.

Widget:

  • Click on the "Fill Contours" button and select one of Never, Always or Except on Overlay. This button rotates through the three options.
Command:
  • FILLTYPE = type
Arguments :
  • type: valid choices for type are :
    • LINES - produces contour line plots
    • SOLID - produces color-filled contour plots with contour line overlays
    • SOLIDALWAYS - produces color-filled contour plots and overlays.
    • PATTERN - produces grayscale stipple plots and overlays.
Examples :
  • FILLTYPE = LINES
Notes :
  • Contour line plots are produced by default.
  • This option remains in effect until changed
