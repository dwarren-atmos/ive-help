This command allows you to select the color with which to display the vectors in a plot.

Widget:

Command:
  • VECTOR_COLOR=color
Arguments:
  • color: the color can be either red, blue, black, green, purple, brown, white, or yellow.
Examples:
  • VECTOR_COLOR=red
Notes:
  • This command does not effect the currently displayed plots.
  • For detailed instructions on creating vector plots, see Producing a Vector Plot.
