This command allows you to select a map projection, and is used in conjunction with the commands PLAT, PLON, and ROTA.

Widget:

  • Select "Base map" from the "Plot Background" menu. When the Map Optons widget appears, use the left mouse button to click-and-hold on the Map Projection cascade menu button. Slide the indicator bar to the desired map projection, then release the mouse button. Note that for the first seven projections listed below, the map limits are gotten from the window limits. Map limits for other projections must be entered via the MAPLIMITS command.
Command:
  • MAPROJ = PROJECTION
Arguments :
  • PROJECTION: valid choices for the map projection are
    • NHST Northern Hemisphere polar stereographic
    • SHST Southern Hemisphere polar stereographic
    • NHOR Northern Hemisphere polar orthographic
    • SHOR Southern Hemisphere polar orthographic
    • ME Mercator
    • CE Cylindrical equidistant
    • MO Mollewide
    • LC Lambert conformal
    • ST Stereographic
    • OR Orthographic
    • LE Lambert equal-area
    • GN Gnomic
    • AE Azimuthal equidistant
    • SV Satellite view
Examples :
  • MAPROJ = ME
    A Mercator projection is used, centered at latitide = 0 and longitude = 0.
Notes :
  • The Default is cylindrical equidistant.
  • The use of this command requires some familiarity with the NCAR Graphics EZMAP routines. For further information on setting the map projection, see the NCAR Graphics documentation on the EZMAP routine MAPROJ.
  • This option remains in effect until changed.
