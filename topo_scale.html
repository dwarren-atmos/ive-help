This command scales the topography contour for 1-D plots.

Widget:

  • There is no widget interface for this command.
Command:
  • TOPO_SCALE=percentage
Arguments:
  • percentage: The percentage of vertical scale of the plot
Examples:
  • TOPO_SCALE=15
    In this case, the topography would be scaled to 15% of the total plot scale.
Notes:
  • The default TOPO_SCALE value is 20%.
