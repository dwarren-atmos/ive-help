This command allows you to lock the color table to specific contour values.

Widget:

  • Select the 'Shift Colors' option from the "Color Table" menu to bring up the Color Options form.
    • To lock -After making your plot press the button associated with the Lock Color Table line (It should say Free). The button will then indicate that the color table is locked.
    • To free -Press the button associated with the Lock Color Table line so that it changes to indicate that the color table is free, or change either the contour interval or the field.
Command:
  • LOCKCOLORTABLE=[free/lock]
  • LOCKCOLORTABLE=contour-value,colortable-entry
Arguments :
  • The following are valid arguments
    • LOCK - locks the colortable based on the current plot.
    • FREE - frees the colortable.
    • CONTOUR-VALUE,COLORTABLE-ENTRY -allows you to link a specific color to a contour level.
Examples :
  • LOCKCOLORTABLE = LOCK
  • LOCKCOLORTABLE = 2.5,40
Notes :
  • Locking requires you to first specify a contour interval and a field (selecting contour values automatically locks the color table).
  • You must also make a plot before locking the colors.
