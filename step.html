This command allows the user to step through IVE log file execution when in the "debug" mode.

Widget:

  • There is no widget interface for this command.
Command:
  • STEP
Arguments:
  • No argument is required.
Example:
  • STEP
Notes:
  • To execute this command, you must be running IVE in the "debug" mode. This is done by typing ive -debug -logfilename at the terminal prompt. The "debug" mode runs the log file a command at a time. Typing "STEP" executes the next command in the log file.
  • To create a log file, refer to LOGFILE.
