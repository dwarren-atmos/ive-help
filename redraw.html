This command allows you to redraw any of the 15 most recent plots.

Widget:

  • There is no widget selector for this command.
Command:
  • REDRAW=number
Arguments:
  • number: the number of the plot you wish to redraw.
Examples:
  • REDRAW=9
Notes:
  • To produce a list of the 15 most recent plots, use the LISTSEGMENTS command.
