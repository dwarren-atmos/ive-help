This command makes it possible to choose contour values for a plot.

Widget:

  • Click the left mouse button on the Values button. Move the cursor to the type-in box and enter the desired contour values, then press <RETURN>.
Command:
  • CONTOUR_VALUES = value, value, value...
Arguments :
  • value: value is the desired contour value
Examples :
  • CONTOUR_VALUES = 0, 10, 15
    The data is contoured at the values 0, 10 and 15
