This command can be used to turn on or off contour plot labels.

Widget:

  • Select the "Annotation" widget from the "Plot Background" menu. use the left mouse button to click on the Other Text / Labels button. The value of this button will toggle between ON and OFF
Command:
  • LABEL = choice
Arguments :
  • choice: choice may be either ON or OFF
Examples :
  • LABEL = OFF
Notes :
  • The default is on.
  • This option remains in effect until changed.
