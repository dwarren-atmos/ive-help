This command allows you to produce contour plots with only a single-valued contour line.

Widget:

  • There is no widget interface for this command.
Command:
  • SINGLEVALUE = [ON / OFF]
Arguments :
  • ON: only the single contour line, whose value is specified by the CONTOUR_INTERVAL command, is drawn.
  • OFF: contours are drawn at the value specified by the CONTOUR_INTERVAL command.
Examples :
  • CONTOUR_INTERVAL = 20, SINGLEVALUE = ON
    plot only the level 20 contour
Notes :
  • This command has been supeseded by the CONTOUR_VALUES command.
