This command specifies the width of the plot border.

Widget:

  • There is no widget interface for this command.
Command:
FRAMEWIDTH=value
Arguments:
  • value: {1,2,3,4}. 1=thinnest; 2=thin; 3=thick; 4=thickest.
Examples:
  • FRAMEWIDTH=2
Notes:
  • The arguments "NORMAL", "THICK" and "THICKEST" are not valid for this command (see LINEWIDTH).
