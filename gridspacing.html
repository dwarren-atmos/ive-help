This command lets the user select the grid line spacing on map projections.

Widget:

  • Enter the desired grid line spacing into the Grid Spacing type-in box and press <RETURN>
Command:
  • GRIDSPACING = spacing
Arguments :
  • spacing: spacing is entered as degrees. Map grid lines will be drawn at the specified interval.
Examples :
  • GRIDSPACING = 20
    grid lines are drawn each 20 degrees latitude and longitude.
Notes :
  • By default grid lines are drawn each 10 degrees latitude and longitude.
  • this option remains in effect until changed.
