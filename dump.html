This command saves the current image in the Graphic Display Window to a file in either XWD or GIF format.

Widget:

  • From the IVE Main Window, select "Output". From the Output menu, select "Dump Window" to activate the Dump Window. On the bottom left corner of the Dump Window, select either XWD or GIF as the graphics file type. Then, specify the directory and file name where you wish to store the file. Finally, click on "OK" to perform the dump.
Command:
  • DUMP=name, type
Arguments:
  • name: the path and file name
  • type: the graphics format you desire. You may select either GIF or XWD as the file type.
Examples:
  • DUMP=~/ive/images/plot.xwd, xwd
Notes:
  • If no graphics file type is specified in the DUMP command, IVE uses GIF as the default format. However, once XWD is chosen as the file type, XWD becomes the new default format for your current IVE session.
