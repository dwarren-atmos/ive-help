This command specifies the Unix command used to send a file to the printer.

Widget:None current. Command:

  • PRINTER_COMMAND=name
Arguments:
  • name: the name of the Unix command used to send a file to the printer. Current supported names are "lpr" (default) and "lp".
Examples:
  • PRINTER_COMMAND=lp
  • PRINTER_COMMAND=lpr
