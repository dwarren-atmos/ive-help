This command allows you to set the limits of the map projection.

Widget:

  • There is currently no widget interface to this command.
Command:
  • MAPLIMITS = LIMITS,LIM1,LIM2,LIM3,LIM4,LIM5,LIM6
ARGUMENTS :
  • LIMITS: specifies whether the arguments lim1-lim6 indicate the limits of the map based on corner points, points on the edge of the map, angles, or the maximum useful area of the map.
  • LIM1-LIM6: arguments specifying the map limits. The meaning of these arguments and the number required are dependent on the value of limits.
Examples :
  • MAPLIMITS = MA
    the maximum useful area produced by the projection is plotted. The argumenss lim1-lim6 are ignored.
Notes :
  • The default is MA.
  • The use of this command requires some familiarity with the NCAR Graphics EZMAP routines. For further information on setting map limits, see the NCAR Graphics documentatain on the EZMAP routine MAPSET.
