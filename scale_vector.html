This command can be used to turn on or off vector scaling

Widget:

  • Button on property widget for vectors

Command:

  • SCALE_VECTOR = choice

Arguments :

  • choice: choice may be either ON or OFF

Examples :

  • SCALE_VECTOR = OFF

Notes :

  • The default is on.
  • This option remains in effect until changed.
  • When on, vectors are scaled in size and direction to match the scale of the axes on the plot.
  • When off, vectors are plotted with the same scale in both directions.
