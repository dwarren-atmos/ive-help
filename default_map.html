This command switches the default map parameters on or off.

Widget:

  • From the Input Menu of the IVE Main Window, select "Load Transforms". This activates the Transform Widget. Now select the default_map transform from the Transform Widget and click on "OK" with the left mouse button.
Command:
  • DEFAULT_MAP=[ON/OFF]
Arguments:
  • ON - activates default map parameters
  • OFF - deactivates default map parameters
Examples:
  • DEFAULT_MAP=ON
Notes:
  • The default map transform specifies such parameters as longitude, latitude, and coriolis effects.
