This command specifies how many decimal places tic labels should display.

Widget:

  • From the IVE Main Window, open the Plot Background Window by clicking on "Plot Background". Then, select "Annotation" to activate the Annotation Window. Finally, in the Decimal Places box, enter the desired value.
Command:
  • TIC_DECIMALS=places
Arguments:
  • places: the number of decimal places to be displayed by the tic labels.
Examples:
  • TIC_DECIMALS=1
    This rounds all tic labels to the nearest tenth.
Notes:
  • The argument must be a small positive integer or zero.
