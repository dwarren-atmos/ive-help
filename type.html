This command specifies the type of field you wish to plot.

Widget:

  • In the IVE Main Window the "type" button displays either "Scalar Plot" or "Vector Plot". Clicking once on the "type" button activates a window containing two choices: "Scalar Plot" and "Vector Plot". Select the desired field type by clicking a second time with the left mouse button.
Command:
  • TYPE=field_type
Arguments:
  • field_type: the two possible arguments are "scalar" or "vector".
Examples:
  • TYPE=vector
Notes:
  • Once the field type is changed, IVE automatically alters the properties window. For scalar plots, you are required to select contour intervals, contour line labels, and the fill type (see CONTOUR_INTERVAL, CONTOUR_VALUES, LINELABELS, NULBLL, and NOZERO.) On the other hand, for vector plots, you must specify the vector display, color, horizontal and vertical components, maximum value, lock, position, scale, and the intervals between vectors (see STREAMLINE, VECTOR_COLOR, VECTOR_COMPONENTS, VECTOR_MAX, VECTOR_LOCK, VECTOR_POSITION, VECTOR_SCALE and VECTOR_INTERVAL).
