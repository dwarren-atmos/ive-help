This command specifies which printer to use if more than one printer is available to your network.

Widget:

  • Select "Output" from the IVE Main Window to open the Output Menu. Then, click on "Print" to bring up the Print Menu. Select Print Setup to activate the Printer Options Window. Finally, with the left-hand mouse button, click on the name of the printer, or type its name in the text box provided. Click on "OK" and "Done" when you are finished.
Command:
  • PRINTER_NAME=name
Arguments:
  • name: the name of the printer in your network that you wish to use, or "default" for your default printer.
Examples:
  • PRINTER_NAME=laser1
  • PRINTER_NAME=default
Notes:
  • When you specify a printer using the PRINTER_NAME command, IVE sends all print commands to that printer until a different printer is specified.
  • If the PRINTER_NAME command is not used, all print commands are directed to the default printer.
