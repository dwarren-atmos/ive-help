This command specifies the projection longitude for your map projection.

Widget:

  • There is no widget interface for this command.
Command:
  • PLON=longitude
Arguments:
  • longitude: the desired projection longitude between -180 and 180 degrees.
Examples:
  • PLON=90
    Designates 90 degrees east as the projection longitude.
Notes:
  • To set the projection latitude, refer to PLON.
  • To set the map rotation, refer to ROTA.
  • For more information on map projections, refer to the manual for NCAR Graphics subroutine "Easy Map".
