This command sets the map rotation angle for your map projection.

Widget:

  • There is no widget interface for this command.
Command:
  • ROTA=angle
Arguments:
  • angle: the rotation angle in degrees.
Examples:
  • ROTA=45
    Rotates the map projection 45 degrees to the east.
Notes:
  • To set the projection longitude, refer to PLON.
  • To set the projection latitude, refer to PLAT.
  • For more information on map projections, refer to the manual for NCAR Graphics subroutine "Easy Map".
