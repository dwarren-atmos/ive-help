This command specifies the maximum number of plots that IVE will retain in memory

Widget:

  • There is no widget interface for this command
Command:
  • MAX_PLOT=value
Arguments:
  • value: The maximum number of plots that IVE will retain in memory. The argument can only be an integer between 1 and 15.
Examples:
  • MAX_PLOT=12
Notes:
  • See also REDRAW and LISTSEGMENTS.
