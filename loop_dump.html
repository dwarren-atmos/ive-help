This command saves a loop as a series of graphics files in GIF or XWD file format.

Widget:

  • See Looping.
Command:
  • LOOP_DUMP=filename.##,EXT
Arguments:
  • filename: the root name of the files
  • EXT: the file extension [GIF, XWD]
Examples:
  • LOOP_DUMP=~/speed.##,GIF saves the loop as the files speed.01, speed.02, speed.03, etc. in GIF format.
Notes:
  • If the path is not specified, the loop will be saved to the directory where IVE was executed.
