To build IVE from source, you need to have a version of make compatible with GNU Make. You also need version 3.2.1 or greater of NCAR Graphics library, a Motif library (which is included with most Unix OS), and the following public domain libraries:

  • NetCDF - version 3.3 or greater
  • udunits - version 1.10.8 or greater
  • xgks - version 2.5.5 with UW modifcations
    (svn co https://www.atmos.washington.edu/svn/xgks-uw)

The makefile assumes that the environment variable NCARG_ROOT has been set, the libraries have been installed in /usr/local/lib, and that the include files are in /usr/local/include. If they are in other locations you will have to edit the config file accordingly. Architecture dependant config files are in the config directory. You should examine the proper config file for your architechture prior to compiling.

IVE has at various times been successfully compiled from source for the following systems:

  • Sun 4 - SunOS 4.1.3
  • Sun - Solaris 2.5 and later with Sun unbundled compilers
  • DEC Alpha - Digital Unix 3.2C and later with DEC compilers
  • DEC MIPS RISC - Ultrix 4.3 with DEC compilers
  • SGI - IRIX 5.3 with SGI compilers
  • Debian Sarge
  • Debian Etch

It is actively used on True64 Unix and Linux at the University of Washington.

If you have a bug, or you wish to contribute other platforms to the IVE distribution, please send mail to ive-bugs@atmos.washington.edu.
