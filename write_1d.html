This command prints the values of the last 1D plot to a file.

Widget:

  • There is no widget interface for this command.
Command:
  • WRITE_1D=filename
Arguments:
  • filename: The pathname of the file where the values will be stored.
Examples:
  • WRITE_1D=theta.out
Notes:
  • WRITE_1D prints the data to the file in a tabulated format.
