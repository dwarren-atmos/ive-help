This command allows you to choose the plot line width.

Widget:

  • From the "Custom Lines" menu select the "Width" submenu. Then select one of: "Both", "Pos", "Neg" or "Default". If you did not select Default you continue on and select a width (see Custom Lines).
Command:
  • LINEWIDTH = type
Arguments :
  • type: valid choices for type are
    • NORMAL - lines of normal width are drawn. These lines are the thinnest possible lines on the display device.
    • THICK - lines are twice as thick as NORMAL.
    • THICKEST - lines are three times as thick as normal.
Examples :
  • LINEWIDTH = THICK
Notes :
  • The actual appearance of the lines will vary depending on the output device.
  • This option remains in effect until changed.
