This command allows you to set the minimum and maximum values on the plot abscissa.

Widget:

  • In the Line Options widget, move to the Horizontal Axis type-in boxes and type in your selection. Press <RETURN> when finished. Note that either the abscissa minimum, or maximum, or both may be set. If either of these values is set to default then IVE will choose an appropriate value.
Command:
  • ABSCISSA = min,max
Arguments :
  • min: the minimum value along the plot abscissa.
  • max: the maximum value along the plot abscissa.
Examples :
  • ABSCISSA = -5,5 the plot abscissa will run from values of -5 to 5. ABSCISSA = ,2 the graphics package will choose an appropriate minimum value (provided the user has not previously set it), and the maximum value will be 2.
Notes :
  • This option remains in effect until changed.
  • Setting either the minimum or maximum to default means that the graphics package will choose an appropriate value.
  • See also HORIZONTAL_SCALE.
