This command locks the vector Maximum Value.

Widget:

  • From the vector Properties Window, click on the box labeled "Locked" to select either "yes" or "no".
Command:
  • VECTOR_LOCK=[ON / OFF]
Arguments:
  • ON - locks the vector Maximum Value
  • OFF - unlocks the vector Maximum Value
Examples:
  • VECTOR_LOCK=ON
Notes:
  • The VECTOR_LOCK command is useful for creating loops of vector plots. The command prevents the Maximum Value from changing during successive plots.
  • For more detailed information, refer to Producing a Vector Plot.
