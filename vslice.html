This command sets an arbitrary vertical slice for a three dimensional data field.

Widget:

  • There is no widget interface for this command.
Command:
  • VSLICE=(X1, Y1),(X2, Y2)
Arguments:
  • (X1, Y1) - one point on the vertical slice.
  • (X2, Y2) - a second point on the vertical slice.
Examples:
  • VSLICE=(0, 0), (300, 300)
Notes:
  • The time coordinate T must be locked for this command to execute.
