For data that includes terrain, this command sets the terrain contour interval

Widget:

  • This command does not currently have a widget interface.
Command:
  • TCONTOUR_INTERVAL = interval
Arguments :
  • interval: interval is the desired contour interval in the same units as the vertical coordinate.
Examples :
  • TCONTOUR_INTERVAL = 0.1
    If the vertical coordinate is in kilometers, this sets the terrain contour interval to 100 meters.
Notes :
  • The default is 0 (the contouring routines choose the interval)
