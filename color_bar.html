This command can be used to turn on or off the color bar on color fill plots.

Widget:

  • Button on the annotation widget. The annotation widget is in the "Plot Backgroud" pull-down menu.

Command:

  • COLOR_BAR = choice

Arguments :

  • choice: choice may be either ON or OFF

Examples :

  • COLOR_BAR = OFF

Notes :

  • The default is on.
  • This option remains in effect until changed.
