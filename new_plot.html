This command erases all plots from the graphic display window and creates a new scalar or vector plot based on the options you have set.

Widget:

  • Click the mouse pointer on the "New Plot" button located in the IVE Main Window.
Command:
  • NEW_PLOT
Examples:
  • NEW_PLOT
Notes:
  • Upon first entering IVE, some other commands must be given prior to the NEW_PLOT command. Which commands must be given will depend on the type of plot you have requested. At a minimum, a data file (FILEcommand), a field (FIELD command), and a plot type (TYPE command) must have previously been given in order to make a plot (see Producing a 2D Scalar Plot, Producing a 1D Scalar Plot, and Producing a Vector Plot for complete instructions).
