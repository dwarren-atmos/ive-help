XPOINT

This command sets the X values in the three points used for arbitrary slicing

Widget:

  • There is no direct widget interface to this command. It is set automatically when Mouse Slicing is used, or derived from the settings in the Arbitrary Slicer widget

Command:

  • XPOINTS=value1,value2,value3

Arguments:

  • value1, value2, value3: optional numeric arguments: set the X value in the corresponding point. These points are used only by the arbitrary slicer.

Examples:

  • XPOINTS=300, 400, 400
  • XPOINTS=,300
  • XPOINTS=300,400
