This command saves your plot as an Encapsulated PostScript (EPS) file.

Widget:

  • From the IVE Main Window, select "Output". Then, from the Output Menu, select EPS to activate the EPS popup. Specify the path and filename where you want to store to the postscipt plot. Make sure you specify whether or not to create the file in color by clicking on the box in the bottom left corner (see also EPS_COLOR).
Command:
  • WRITE_FRAME=filename
Arguments:
  • filename: the directory and file name where you wish to store the postscipt file
Examples:
  • WRITE_FRAME=~/ive/images/plot1.eps
Notes:
  • The EPS file format is basically a set of printer commands. It is a format that is compatible with most printers.
