ZPOINT

This command sets the Z values in the three points used for arbitrary slicing

Widget:

  • There is no direct widget interface to this command. It is set automatically when Mouse Slicing is used, or derived from the settings in the Arbitrary Slicer widget

Command:

  • ZPOINTS=value1,value2,value3

Arguments:

  • value1, value2, value3: optional numeric arguments: set the Z value in the corresponding point. These points are used only by the arbitrary slicer.

Examples:

  • ZPOINTS=300, 400, 400
  • ZPOINTS=,300
  • ZPOINTS=300,400
