This command allows you to control tic label style.

Widget:

  • From the IVE Main Window, open the Plot Background Window by clicking on "Plot Background". Then, select "Annotation" to activate the Annotation Window. Finally, click on either "Off", "Normal", or "Bold" as the tic style.
Command:
  • TIC_LABELS=style
Arguments:
  • style: Three tic label styles are available
    • OFF - tic labels disabled in all successive plots
    • NORMAL - tic labels enabled in all successive plots
    • BOLD - bold tic labels enabled in all successive plots
Examples:
  • TIC_LABELS=OFF
Notes:
  • Changes to tic labels do not take effect until a new plot is generated.
  • Only major tics are labeled in IVE plots.
