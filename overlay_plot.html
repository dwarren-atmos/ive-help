This command overlays an existing scalar or vector plot.

Widget:

  • From the IVE Main Window, click on "Overlay Plot" with the left mouse button.
Command:
  • OVERLAY_PLOT
Examples:
  • OVERLAY_PLOT
Notes:
  • A NEW_PLOT must be produced first before executing this command.
  • Any combination of scalar and vector overlays is possible.
  • You may create color-filled overlays by selecting the "Always" Color Fill option from the Properties Window (see also FILLTYPE).
