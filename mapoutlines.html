This command allows you to turn off the display of continental outlines on map projections.

Widget:

  • Select "Base Map" from the "Plot Background" menu. When the Map Optons widget appears, use the left mouse button to click on the Continental Outlines toggle button. The value of this button will toggle between YES (continental outlines will be drawn on a map background) and NO (contenental outlines will not be drawn).
Command:
  • MAPOUTLINES = outline_flag
Arguments :
  • outline_flag: valid choices for outline_flag are
    • CO - continental outlines
    • US - U.S. state outlines
    • PO - continental and international outlines
    • PS - US and PO
    • M3 - North American outlines with 3 mile resolution
    • M1 - North American outlines with 1 mile resolution
    • NO - no map outlines will be drawn on maps.
Examples :
  • MAPOUTLINES = CO
Notes :
  • The default is CO.
  • This option remains in effect until changed.
