This command allows you to print contour plots in color.

Widget:

  • Select "Output" from the IVE Main Window to open the Output Menu. Then, click on "Print" to bring up the Print Menu. Select "Print Setup" to activate the Printer Options Window. Finally, choose "Yes" or "No" from the box labeled "Color Lines?" by clicking with the left mouse button.
Command:
  • PRINT_COLOR=[ON / OFF]
Arguments:
  • ON - all contour plots printed in color
  • OFF - all contour plots printed in black and white
Examples:
  • PRINT_COLOR=ON
Notes:
  • Although IVE contour plots are displayed in color on the screen, print commands are sent to the printer in black and white unless otherwise specified.
  • This command only applies to contour plots.
