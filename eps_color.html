This command specifies whether or not to create Encapsulated PostScript (EPS) files in color.

Widget:

  • From the IVE Main Window, select "Output". Then, from the Output Menu, select EPS to activate the EPS popup. Finally, click on the "Color lines" box to select either "Yes" or "No".
Command:
  • EPS_COLOR=[ON / OFF]
Arguments:
  • ON - all successive EPS files will contain color lines
  • OFF - all successive EPS files will be created in black and white
Examples:
  • EPS_COLOR=ON
Notes:
  • The EPS file format is basically a set of printer commands. It is a format that is compatible with most printers.
