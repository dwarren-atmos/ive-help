This command specifies the position of vectors relative to their gridpoints.

Widget:

  • From the vector Properties Window, select the vector position by clicking on the box labeled "Position of grid point on vector".
Command:
  • VECTOR_POSITION=position
Arguments:
  • position: indicates the position of the vectors relative to the gridpoints. "Tail", "head", and "center" are the three possible arguments.
Examples:
  • VECTOR_POSITION=tail
Notes:
  • Execution of this command does not alter the vector position of currently displayed vector plots.
  • Detailed instructions for creating vector plots are available in the document "Producing a Vector Plot".
