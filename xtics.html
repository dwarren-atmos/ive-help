This command specifies the major and minor tic intervals along the abscissa axis in your plot.

Widget:

  • At the Properties Window, input the desired major tic interval and minor tic interval values in the boxes provided for the abscissa axis.
Command:
  • XTICS=major, minor
Arguments:
  • major: the value of each major tic interval
  • minor: the value of each minor tic interval
Examples:
  • XTICS=100, 25
Notes:
  • This command applies to both one and two-dimensional plots.
